# Contributing

Contributions are welcome.

Please keep the extension's single purpose narrow: user-selected video codec capability blocking on user-selected websites.

Before submitting a pull request:

1. Test the extension with `chrome://extensions` > Developer mode > Load unpacked > `extension/`.
2. Verify that no remote code, tracking, analytics, or unrelated permissions were added.
3. Update `CHANGELOG.md` and documentation if behavior changes.
4. Keep user-facing strings available in both English and Traditional Chinese.
