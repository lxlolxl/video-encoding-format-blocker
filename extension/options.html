Video Encoding Format Blocker

Block selected video codec capability reporting on chosen websites.

Blocked video formats

Default is AV1 only. Blocking too many formats may force lower quality or make some videos unavailable.

Target websites

Use Chrome match patterns, one per line.
*://*.netflix.com/* *://*.youtube.com/*

Permission note

When saving, Chrome may ask for permission to run on the patterns you entered. The extension only registers scripts on the sites listed above.
