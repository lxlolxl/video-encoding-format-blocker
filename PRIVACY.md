# Privacy Policy

Video Encoding Format Blocker does not collect, sell, transmit, or share personal data.

## Data stored locally

The extension stores these settings using Chrome extension storage:

- Whether the blocker is enabled.
- The user's selected website match patterns.
- The user's selected blocked formats: AV1, VP9, and H.265 / HEVC.
- The user's UI language preference.

These settings are used only to register the extension's content script on the websites selected by the user.

## Network access

The extension does not make external network requests.

## Website data

The extension does not read page content, browsing history, cookies, passwords, form inputs, video content, or account information. It only overrides media capability detection APIs on user-selected websites so that selected codecs are reported as unsupported.

## Limited Use disclosure

The extension's use of browser and website access is limited to its single purpose: allowing users to block selected video encoding format capability reporting on websites they choose. The extension does not use data for advertising, tracking, profiling, or unrelated purposes.
