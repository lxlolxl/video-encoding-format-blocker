# Chrome Web Store Privacy Questionnaire Draft

This draft should be reviewed against the current Chrome Web Store dashboard form before submission.

## Single purpose

Allow users to block selected video encoding format capability reporting on websites they choose.

## Data collection

The extension does not collect, transmit, sell, or share user data.

## Personally identifiable information

No.

## Health information

No.

## Financial and payment information

No.

## Authentication information

No.

## Personal communications

No.

## Location

No.

## Web history

No. The extension stores only user-entered website match patterns and does not read browsing history.

## User activity

No. The extension does not track clicks, browsing behavior, playback behavior, or page content.

## Website content

No. The extension does not collect page content. It only overrides media capability APIs on user-selected websites.

## Limited Use

The extension's use of browser access is limited to the disclosed single purpose. It does not use data for advertising, tracking, profiling, or unrelated purposes.
